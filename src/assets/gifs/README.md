# Quest Activity GIFs

Place your pet-specific GIF files in these directories:

## Meditation GIFs (for Mindful Moment quest):
- `meditation/dog.gif` - Dog meditation GIF
- `meditation/cat.gif` - Cat meditation GIF  
- `meditation/rabbit.gif` - Rabbit meditation GIF
- `meditation/penguin.gif` - Penguin meditation GIF

## Exercise GIFs (for Move Your Body quest):
- `exercise/dog.gif` - Dog exercise GIF
- `exercise/cat.gif` - Cat exercise GIF
- `exercise/rabbit.gif` - Rabbit exercise GIF
- `exercise/penguin.gif` - Penguin exercise GIF

## File Requirements:
- Format: GIF
- Recommended size: 256x256 pixels or similar square aspect ratio
- Keep file sizes reasonable for web loading

## Usage:
The GIFs will be displayed above the timers during the respective quest activities based on the user's selected pet type.
