# Quest Activity GIFs

Place your pet-specific GIF files in the **public** directory:

## Directory Structure:
```
public/gifs/
├── meditation/
│   ├── dog.gif
│   ├── cat.gif
│   ├── rabbit.gif
│   └── penguin.gif
└── exercise/
    ├── dog.gif
    ├── cat.gif
    ├── rabbit.gif
    └── penguin.gif
```

## File Requirements:
- Format: GIF
- Recommended size: 256x256 pixels or similar square aspect ratio
- Keep file sizes reasonable for web loading

## Usage:
The GIFs will be displayed above the timers during the respective quest activities based on the user's selected pet type.

**Important:** Move your GIF files from `src/assets/gifs/` to `public/gifs/` for them to display correctly.
